<iisapppool>

Allows an IIS application pool to be controlled.

Parameters

Attribute Type Description Required
action ApplicationPoolAction The action that should be performed on the application pool. True
pool string The name of the application pool on which to perform the action. True
server string The name of the server on which to perform the action. The default is the local computer. True
failonerror bool Determines if task failure stops the build, or is just reported. The default is true. False
if bool If true then the task will be executed; otherwise, skipped. The default is true. False
unless bool Opposite of if. If false then the task will be executed; otherwise, skipped. The default is false. False
verbose bool Determines whether the task should report detailed build log messages. The default is false. False

Examples

  • Starts the "StsAdminAppPool" application pool on server "SV-ARD-WEB". 

    <iisapppool action="Start" pool="StsAdminPool" server="SV-ARD-WEB" />

  • Stops and restarts the "DefaultAppPool" application pool on server "SV-ARD-WEB". 

    <iisapppool action="Restart" pool="DefaultAppPool" server="SV-ARD-WEB" />
